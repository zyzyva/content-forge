After you have your idea and hook, this will help you build a viral script. Custom built and used by the Hard Knocks Team.

**Link to GPT:** https://chatgpt.com/g/g-68e5b5eb45008191aee05e360182ea29-viral-script-lab

### STEP 1: Lock In Your Voice

At the start of a new chat, paste your completed *My Voice Prompt*. You can find this in the Prompt Library.

New Chat = Memory Reset. You must upload Your Voice Prompt each time you start a new chat.

### STEP 2: Build Your Prompt

Copy the *Viral Script Prompt* in the Prompt Library into a document for you to save as a reference later, and edit the section in bold that is marked with the customization marker.

### STEP 3: Enter Your Prompt

Paste the completed prompt into the Viral Script Machine GPT. If you prefer to use another AI tool, you may do so.
